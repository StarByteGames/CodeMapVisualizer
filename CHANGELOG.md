## [0.0.1] - 2026-01-29
### Added
- Initial release of CodeMapVisualizer.
- Automatically parses C# source code to generate UML diagrams.
- Displays interactive UML diagrams in a dedicated VS Code webview.
- Integrates with the activity bar for quick access.
- Command: "CodeMapVisualizer: UML-Diagramm anzeigen" to show diagrams.
